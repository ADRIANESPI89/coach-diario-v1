# Coach Diario — Visión Visual

Versión: v1

---

# Objetivo

Definir cómo debe sentirse Coach Diario.

---

# Principio

La interfaz debe desaparecer.

La persona debe sentir espacio.

---

# Vista Día

Concepto:

Espacio abierto

Sensación:

- claridad
- aire
- simpleza

Evitar:

- saturación
- exceso de color
- demasiados elementos

La interfaz debe invitar.

---

# Vista Noche

Concepto:

Descanso visual

Sensación:

- calma
- silencio
- presencia

Paleta:

- negro profundo
- azul suave

Evitar:

- blancos intensos
- brillos
- contraste agresivo

---

# Regla

Cada pantalla debe responder:

¿Se entiende?

¿Descansa la vista?

¿Invita a quedarse?

---

# Filosofía

Menos interfaz.

Más espacio.
